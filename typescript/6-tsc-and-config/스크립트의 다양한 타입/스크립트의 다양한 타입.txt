함수 타입

함수의 파라미터와 반환값에 타입 지정
예상치 못한 타입의 인자가 전달 되거나 타입의 값이 반환되는 것을 방지
삼수 시그니처로 해당 함수의 사용법을 자련스럽게 문서화

파라미터 타입: 함수에 전달하는 매개변수의 데이터타입을 지정하며, 올바른 개수가
전달되는지 또한 확인한다

옵셔널 파라미터 타입: 삼수에 필수가 아닌 선택적 매개변수를 지정할 수 있다.

콘텍쳐 타입: 함수(콜백 함수)의 맥락에 따라 자동으로 타입을 추론하는 기능

리턴 타입: 함수가 반환하는 값의 데이터 타입을 명시

보이드&네버 타입
보이드 반환 타입: 함수가 명시적으로 값을 반환하지 않음
네버 반환 타입: 함수가 값을 반환하지 않음을 엄격하게 표현

유니언 타입
여러 타입 중 하나가 될 수 있는 값을 표현
집한 이론의 "합집합"과 동일한 개념 (A U B)
서로 다른 타입의 값을 하나의 변수에 할당할 수 있으면 "|"를 사용하여 나타낸다

type narrowing:
유니언 타입의 구체적인 하위 타입을 좁혀나가는 방식('타입 좁히기'라고도 표현한다)
배열:
서로 다른 타입의 요소를 포함할 수 있는 배열을 정의하여
유연성을 증가시킨다.
리터럴:
제한된 값을 가진 타입(특정 문자열이나 숫자의 집합을 표현)
